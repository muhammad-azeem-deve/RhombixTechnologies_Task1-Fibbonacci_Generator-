# RhombixTechnologies_Task1-Fibbonacci_Generator-
Built a Fibonacci Generator using Python, Flask, and Tailwind CSS  This project combines backend logic with a clean, responsive UI to generate Fibonacci sequences dynamically. A great hands-on experience in full-stack development, integrating logic, APIs, and modern frontend styling.

A simple and interactive **Fibonacci Generator** built using **Python (Flask)** for backend processing and **Tailwind CSS** for a modern, responsive UI. This project demonstrates full-stack integration with clean logic and user-friendly design.

---

## 📌 Project Overview

This web application allows users to input a number and dynamically generates the Fibonacci sequence up to that limit. It combines backend computation with a sleek frontend interface.

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, Tailwind CSS

# 5.  How It Works

User inputs a number (n)
Flask processes the request
Backend generates Fibonacci sequence up to n terms
Result is displayed dynamically on the UI
